# 公文正文排版参数手册

> 依据：GB/T 9704-2012《党政机关公文格式》

| 参数 | 值 |
|------|-----|
| 纸张 | A4 (210mm × 297mm) |
| 上边距 | 37mm |
| 下边距 | 28mm |

| 右边距 | 26mm |
| 行间距 | 固定值 28 磅 |

## 正文层级字体

| 层级 | 序数格式 | 正则模式 | 字体 | 字号 | 加粗 |
|------|---------|---------|------|------|------|
| 标题 | — | — | 方正小标宋简体 | 二号 (22pt) | 否 |
| 一级 | `一、` `二、` | `^[一二三四五六七八九十]+、` | 黑体 | 三号 (16pt) | 否 |
| 二级 | `（一）` `(一)` | `^[（(][一二三四五六七八九十]+[）)]` | 楷体_GB2312 | 三号 (16pt) | 否 |
| 三级 | `1.` `1、` | `^\d+[.、]` | 仿宋_GB2312 | 三号 (16pt) | 否 |
| 四级 | `(1)` `（1）` | `^[（(]\d+[）)]` | 仿宋_GB2312 | 三号 (16pt) | 否 |
| 正文 | — | — | 仿宋_GB2312 | 三号 (16pt) | 否 |

### 字号对照

| 中文号数 | 磅值 (pt) |
|---------|-----------|
| 二号 | 22pt |
| 三号 | 16pt |
| 四号 | 14pt |

### 层级匹配规则

按顺序依次尝试匹配，命中即停止：

1. `^[一二三四五六七八九十]+、` → 一级标题（黑体）
2. `^[（(][一二三四五六七八九十]+[）)]` → 二级标题（楷体_GB2312）
3. `^\d+[.、]` → 三级标题（仿宋_GB2312）
4. `^[（(]\d+[）)]` → 四级标题（仿宋_GB2312）
5. 以上均不匹配 → 正文

全角/半角括号均支持。三级标题支持 `1.` 和 `1、` 两种序号格式。

### 数字与字母

全文数字和英文字母统一使用 **Times New Roman**。脚本应在写入每段时自动处理中英文混排的字体切换。

## 附件格式

- 正文结束后空一行
- "附件" 二字左空两格（缩进 2 字符），后跟附件列表序号
- 每个附件名称单独一行，格式：`附件：1. xxx`
- 附件列表结束后空两行（留给办公室添加落款）

## 图片处理

- 原文中的嵌入图片自动提取，插入到段落间对应位置
- 图片居中，独立段落，首行不缩进
- 宽度不超过版心 156mm（A4 210mm - 左28mm - 右26mm），等比缩放
- 支持的格式：`.docx` 中的 `<w:drawing>` 和 `<w:pict>`（VML），`.doc`（通过 Word COM 转换）
- EMF/WMF 矢量图自动转为 PNG

## 表格处理

- 原文中的表格按顺序提取，插入到段落间对应位置
- 表头行：黑体 五号（10.5pt），居中
- 数据行：仿宋_GB2312 五号（10.5pt）
- 表格边框：Table Grid 样式（全边框）
- 表格居中，表后空一行

## 称谓 / 署名 / 日期格式

### 称谓（行文对象）

- 顶格左对齐，**不缩进**
- 字体：仿宋_GB2312 三号
- 位于标题之后、正文之前

### 署名（行文单位）

- **右对齐**，无右缩进
- 字体：仿宋_GB2312 三号
- 位于附件之后，与正文/附件之间空两行

### 日期（成文日期）

- **右对齐 + 右空两字**（`right_indent = 32pt`）
- 字体：仿宋_GB2312 三号
- 位于署名正下方，与署名同行距

## 标题字体分割

一级（`一、`）和二级（`（一）`）标题段落中，若在序号后出现 `：` 或 `。`，则：

- **前缀**（序号至第一个 `：`/`。`）：标题字体（黑体/楷体_GB2312）
- **后缀**（`：`/`。` 之后）：仿宋_GB2312（正文字体）

约束：序号后至少 2 个字符才触发分割，避免短标题误分。

示例：
```
（一）线下社会渠道：通过该功能快速查询…
├─ 前缀（楷体_GB2312）────┤├─ 后缀（仿宋_GB2312）───┤
```

三、四级标题（`1.` / `(1)`）不做字体分割。

## 字体获取与回退

### 国标字体

| 字体名 | 文件名 (Windows) | 来源 |
|--------|-----------------|------|
| 方正小标宋简体 | FZXiaoBiaoSong-B05S.ttf | 方正字库（商用需授权） |
| 仿宋_GB2312 | FANGSONG_GB2312.TTF | Windows 系统自带 |
| 楷体_GB2312 | KAITI_GB2312.TTF | Windows 系统自带 |
| 黑体 | SIMHEI.TTF | Windows 系统自带 |

### 回退策略

生成文档时，脚本依次尝试：

1. 系统已安装字体
2. `--font-dir` 参数指定的目录中的字体文件
3. 字体回退映射

| 目标字体 | 回退字体 |
|---------|---------|
| 方正小标宋简体 | 宋体 (SimSun) |
| 仿宋_GB2312 | 仿宋 (FangSong) |
| 楷体_GB2312 | 楷体 (KaiTi) |
| 黑体 | SimHei |

每个回退产生一条 warning，在 stdout JSON 中返回。

## 特殊排版规则

### 标点与空格

- 三级标题 `1.` 后跟一个半角空格，再跟正文内容
- 四级标题 `(1)` 后直接跟正文内容，不加空格
- 中文标点使用全角（，、。；：？！）
- 英文标点使用半角

### 段落缩进

- **所有段落**（含层级标题和正文）首行左空 2 字符（`paragraph_format.first_line_indent = Pt(32)`）
- 称谓（行文对象）**不缩进**（顶格左对齐）
- 署名/日期使用右对齐，不设首行缩进
- 回行顶格

### 页码

- 日常呈文：`— 1 —` 宋体四号，页脚居中
- 正式行文：页码页脚外侧（单页右/双页左），左右缩进 1 字符（未实现）
